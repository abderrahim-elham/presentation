Introduction

Une petite recherche sur Internet et les sites de réseaux sociaux révèle clairement que de nombreuses établissements , qu'elles soient nationales ou internationales, Très intéressé à utiliser la technologie numérique pour mieux gérer les interactions entre les différents acteurs au sein d'une organisation spécialement au niveau du scolarité, et dans le même cadre, le but de notre projet de fin d’étude qui consiste à la conception et la réalisation d’un site web pour la gestion des exercices Ce qui est l'une des choses les plus importantes dans chaque établissement d'enseignement.

3
Qu'est-ce qu'une plateforme génération des exercices?
Introduction

Plateforme de génération des exercices est une plateforme qui génères plusieurs type d'exercices, alors que deux catégorie de personnes peuvent projeter de cette plateforme soit enseignent soit étudiant.

4
Qu'est-ce qu'une plateforme génération des exercices?
Les objectifs
  • La facilite de rechercher les exercices qui veut quelque soit le domaine

  • Permet aux enseignants de mettre en ligne leurs exercices

  • Création d’un pont de communication entre enseignant et étudiant.

5
Qu'est-ce qu'une plateforme génération des exercices?
Les acteurs

Administrateur : C’est un rôle qui peut être joué par la personne qui est chargée de l’administration de la base de données.

L’enseignant : C’est un rôle qui peut être joué par tous les enseignants du département.

L’étudiant : C’est un rôle qui peut être joué par tous les étudiants du département.

6
Qu'est-ce qu'une plateforme génération des exercices?
Les acteurs : L'administrateur
  • Gérer les enseignants (supprimer, valider l’inscrit)
  • Gérer les étudiants (supprimer, valider l’inscrit).
  • Gérer les modules (supprimer, valider l’inscrit).
  • Gérer les semestres.
7
Qu'est-ce qu'une plateforme génération des exercices?
Les acteurs : L’enseignant
  • inscrire.
  • S’authentifier.
  • générer des exercices.
  • Ajouter des chapitres.
  • Télécharger des exercices.
  • insérer des exercices.
  • contacter l'administrateur et les étudiants.
8
Qu'est-ce qu'une plateforme génération des exercices?
Les acteurs : L’étudiant
  • Inscrire.
  • S’authentifier.
  • Générer des exercices.
  • Télécharger des exercices.
  • Contacter l'administrateur.

jQuery

jQuery est une bibliothèque JavaScript concise et rapide qui peut être utilisée pour simplifier la gestion des événements, la traversée de documents HTML, les interactions Ajax et l'animation pour un développement de site Web rapide. jQuery simplifie les scripts côté client du HTML, simplifiant ainsi le développement d'applications Web 2.0.

La plateforme GénéExercice.

AJAX

AJAX (Asynchrones JavaScript And XML) est une méthode de développement web basée sur l'utilisation d'un script JavaScript pour effectuer des requêtes web à l'intérieur d'une page web sans recharger la page. AJAX rend plus interactif les sites web et offre une meilleure ergonomie ainsi qu'une réactivité améliorée en permettant de modifier interactivement une partie de l'interface web seulement.

Python

Python est le langage de programmation open source le plus employé par les informaticiens. Ce langage s’est propulsé en tête de la gestion d’infrastructure, d’analyse de données ou dans le domaine du développement de logiciels. En effet, parmi ses qualités, Python permet notamment aux développeurs de se concentrer sur ce qu’ils font plutôt que sur la manière dont ils le font. Il a libéré les développeurs des contraintes de formes qui occupaient leur temps avec les langages plus anciens. Ainsi, développer du code avec Python est plus rapide qu’avec d’autres langages.

Django

Django est un framework web basé sur le langage de programmation Python, Sa première version publique et open source est rendue disponible en 2005. Il permet de construire toutes sortes de aplication web (réseaux sociaux, sites d’actualités, etc…) avec toutes les bases nécessaires, telles que la sécurité des applications, mises à disposition dès le départ pour permettre aux développeurs de se concentrer pleinement sur l’essentiel de leurs projets. Il est utilisé par de nombreux sites web tels que Mozilla et Instagram.

MySQL

MySQL est un serveur de bases de données relationnelles Open Source. Un serveur de bases de données qui stocke les données dans des tables séparées plutôt que de tout rassembler dans une seule table. Cela améliore la rapidité et la souplesse de l'ensemble.
